##2、递归的实例
### 求阶乘的递归
```
  function jc(num){

     if(num == 1 ){
        //此处为必须的出口
        return 1
      }
      return num*jc(num-1)
  }
/以上为阶乘的递归写法
```

### 为求和的递归
```
function sum(num){
    if(num == 1){
        //此处为出口
        return 1
     }
     return num  +  sum(num - 1)
}
/求和
```
